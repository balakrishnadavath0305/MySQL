**Introduction to DDL**

Part of SQL to define and manage database objects.

It includes the following commands:



i)Drop: removes the entire structure.

ii)Rename: change the name of the database objects.

iii)Create: used to create database and database objects like Tables, view, and indexes.

iv)Alter: to change the structure of the existing database objects. We can add/drop columns, change datatypes, and rename the column name.

v)Truncate: deletes all the data in the table without affecting the structure.
