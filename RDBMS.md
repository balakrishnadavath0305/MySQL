**What is Relational Database?**

Relational Database is a type of database that stores data in the form of tables in a structured way.



*Characteristics:*

i)Structured

ii)Primary Key

iii)Relationships

iv)Primary key

v)ACID properties

vi)Integrity and Constraints



*Key Concepts of Relational DB*

i)Relations (Tables)

ii)Entities and Attributes

iii)Primary key

iv)Foreign key

v)Normalization

vi)Denormalization



*Design Process of Relational DB*

Need to maintain all the properties mentioned above



*Key Components of RDBMS*

i)Database Engine

ii)Query processor

iii)Storage manager

iv)Transaction Manager
