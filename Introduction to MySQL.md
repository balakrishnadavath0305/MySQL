**Introduction to MySQL**

MySQL is a popular open source Relational Database Management System(RDBMS).



*Features:*

i)Cross Platform

ii)High Performance

iii)Scalability



*MySQL Architecture:*

MySQL Architecture is modular and comprises of:

i)Client

ii)Query optimizer

iii)Storage engines

iv)Pluggable components



it has three key layers:

i)Storage layer

ii)Management layer

iii)Connection layer



*MySQL storage engines:*

i)InnoDB: default, supports transactions

ii)MyISAM: Read intensive, doesn't support transactions

iii)Memory Engine: ultra fast operations

iv)CSV, Archive etc



*Query Execution process:*

i)Client request

ii)SQL parser

iii)Query optimizer

iv)Storage engine

v)Result return



*Replication and Backup:*

*Replication:*

Copying data from MySQL server to other

*Types:*

i)Master-slave

ii)Master-Master



*Backup:*

i)Logical backup

ii)Physical backup
